# Dataset

This project uses datasets originally provided by the **DeepMatcher** repository for research on product matching.

The datasets were originally introduced in the paper:

*Deep Learning for Entity Matching: A Design Space Exploration*, published in SIGMOD 2018.

More information about the datasets can be found in the original DeepMatcher repository:

https://github.com/anhaidgroup/deepmatcher
